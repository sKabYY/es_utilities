Quick start

Let's start from a simple example:

Example 1. I have a lot of logs for Sendmail and I want to search the 10 most frequent deffered domains.

If I use the official RESTful API of Elasticsearch, I can type the following command: 

curl 'my.elasticsearch.host:9200/maillog*' -d '{
    "facets": {"domain0": {"terms": {"field": "@fields.toDomain",
                                      "order": "count",
                                      "size": 10}}},
    "query": {"term": {"@fields.status": "deffered"}},
    "size": 0}'

I think the query is not complicated. But I have to type a very long (and multiple lines) command.

The factors for this query

  1. The field: the field is “@fields.toDomain”.

  2. The size: 10.

  3. Condition: The content of field “@field.status” is “deffered”.
